# Hiring Plan

## Philosophy
Stay as small as possible for as long as possible. Every hire must be revenue-justified. Prefer contractors over FT until $25K MRR.

---

## Hire #1: Developer Marketing Contractor (Month 4-6)
**When**: $5K MRR sustained for 2+ months  
**Budget**: $2-4K/mo  
**Type**: 1099 contractor, 10-15 hrs/week

### What they do
- Write technical blog posts about LLM inference, benchmark methodology, API cost optimization
- Manage Twitter/X presence
- Post in communities (HN, Reddit, Discord)
- Repurpose benchmark data into shareable content

### What you keep
- All product decisions
- All engineering
- All investor/partner relationships

### Job description template
```
We're looking for a developer-focused content creator to grow Tokenomicon's 
audience. You should understand AI APIs, be able to write technical content 
that developers actually read, and enjoy the intersection of code and community.

This is a part-time contractor role ($X/mo, ~10hrs/week). Async-first. 

You'll own:
- 2-3 technical posts/week (Twitter threads, blog posts, Reddit)
- Community presence in AI dev communities
- Benchmark data storytelling

You should have:
- Active presence in AI dev communities
- Writing portfolio with technical content
- Understanding of LLM APIs

Apply: send 3 examples of your best technical content to founder@tokenomicon.io
```

## Interview process (lightweight)

1. Async: 3 work samples + 15-min Loom intro
2. 30-min video call: culture + role fit
3. Paid test project (1 week, $500-1K)
4. Offer

No whiteboard coding. No algorithm puzzles. Real work test only.
